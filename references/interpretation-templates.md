# 各类分析结果解读模板

## t 检验结果解读

### 输出表格关键列
- **Levene's Test for Equality of Variances**: 判断方差齐性
  - Sig. > .05 → 方差齐性，读"Equal variances assumed"行
  - Sig. ≤ .05 → 方差不齐，读"Equal variances not assumed"行
- **t**: t 统计量
- **df**: 自由度
- **Sig. (2-tailed)**: 双尾 p 值
- **Mean Difference**: 均值差
- **95% Confidence Interval of the Difference**: 均值差的 95% CI

### 解读模板

```
Levene 检验结果:
- F = [值], p = [值]
- 判断: [p > .05, 方差齐性满足 / p ≤ .05, 方差不齐]

t 检验结果 (选择对应行):
- t([df]) = [t值], p = [p值]
- 均值差 = [差值]
- 95% CI [下限, 上限]

解读:
[组A] 的 [变量] 得分 (M = [均值], SD = [标准差]) [显著高于/低于/与]
[组B] (M = [均值], SD = [标准差])。

效应量:
- Cohen's d = [值] → [小/中/大]效应

结论:
[用通俗语言说明差异的实际意义]
```

### Cohen's d 计算
- d = 均值差 / 合并标准差
- 小效应 d = 0.2，中效应 d = 0.5，大效应 d = 0.8

## ANOVA 结果解读

### 输出表格关键内容
- **Test of Homogeneity of Variances**: Levene 方差齐性检验
- **ANOVA 表**: F 值、df、显著性
- **Post Hoc 多重比较**: 各组两两比较
- **Descriptives**: 各组描述统计

### 解读模板

```
前提检验:
- 方差齐性: Levene's F([df1], [df2]) = [值], p = [值]
  → [满足/不满足] 方差齐性假设

ANOVA 结果:
- F([df1], [df2]) = [F值], p = [p值]
- 效应量: η² = [值] → [小/中/大]效应

判断: [p < .05, 组间差异显著 / p ≥ .05, 组间差异不显著]

事后检验 (若显著):
- [组A] vs [组B]: 均值差 = [值], p = [值] → [显著/不显著]
- [组A] vs [组C]: 均值差 = [值], p = [值] → [显著/不显著]
- [组B] vs [组C]: 均值差 = [值], p = [值] → [显著/不显著]

结论:
[哪些组之间存在显著差异，差异方向是什么]
```

## 信度分析解读

### 解读模板

```
整体信度:
- Cronbach's α = [值]
- 判断: [>.9 极好 / .8-.9 良好 / .7-.8 可接受 / <.7 需改进]

各题项分析:
| 题项 | 删除后α | CITC | 判断 |
|------|---------|------|------|
| item1 | .xx | .xx | [保留/考虑删除] |
| item2 | .xx | .xx | [保留/考虑删除] |

判断标准:
- CITC (校正项总相关) > .30 → 题项与整体一致
- 删除后α < 整体α → 该题项不降低信度
- CITC < .30 或 删除后α 明显上升 → 建议删除该题项

结论:
[量表信度是否达标，是否有题项需要删除或修改]
```
