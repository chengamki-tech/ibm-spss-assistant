# 统计检验选择决策树

## 第一步：确定研究目的

```
你的研究目的是什么？
├── 比较差异 → 跳到"第二步A"
├── 探索关联 → 跳到"第二步B"
├── 预测结果 → 跳到"第二步C"
├── 降维/结构 → 跳到"第二步D"
└── 因果路径 → 跳到"第二步E"
```

## 第二步A：比较差异

```
有几个组？
├── 1 组 → 单样本 t 检验 (与已知值比较)
│
├── 2 组
│   ├── 独立样本?
│   │   ├── 连续 DV + 正态 → 独立样本 t 检验
│   │   ├── 连续 DV + 非正态 → Mann-Whitney U
│   │   ├── 有序 DV → Mann-Whitney U
│   │   └── 分类 DV → 卡方检验 / Fisher 精确检验
│   └── 配对样本?
│       ├── 连续 DV + 正态 → 配对样本 t 检验
│       ├── 连续 DV + 非正态 → Wilcoxon 符号秩
│       └── 分类 DV → McNemar 检验
│
├── 3+ 组 (独立)
│   ├── 1 个 DV
│   │   ├── 正态 + 方差齐 → 单因素 ANOVA
│   │   ├── 正态 + 方差不齐 → Welch ANOVA
│   │   └── 非正态 → Kruskal-Wallis
│   └── 2+ 个 DV (同时检验)
│       └── 多元方差分析 (MANOVA)
│
└── 3+ 组 (配对/重复测量)
    ├── 正态 → 重复测量 ANOVA (检查球形假设)
    └── 非正态 → Friedman 检验
```

### 事后检验选择

| 场景 | 推荐方法 | 说明 |
|------|---------|------|
| 各组样本量相等 | Tukey HSD | 最常用，平衡型 |
| 各组样本量不等 | Bonferroni | 最保守，适合组数少 |
| 只与一个对照组比较 | Dunnett | 实验组 vs 对照组 |
| 方差不齐 | Games-Howell | 不要求方差齐性 |
| 重复测量 | Bonferroni | 配对比较校正 |

### 非参数替代速查

| 参数方法 | 非参数替代 | SPSS 路径 |
|---------|-----------|-----------|
| 独立 t 检验 | Mann-Whitney U | Nonparametric → Legacy → 2 Independent |
| 配对 t 检验 | Wilcoxon 符号秩 | Nonparametric → Legacy → 2 Related |
| 单因素 ANOVA | Kruskal-Wallis | Nonparametric → Legacy → K Independent |
| 重复测量 ANOVA | Friedman | Nonparametric → Legacy → K Related |

## 第二步B：探索关联

```
两个变量的类型？
├── 连续 × 连续
│   ├── 双变量正态 → Pearson 相关
│   └── 非正态/有异常值 → Spearman 秩相关
├── 有序 × 有序
│   └── Kendall tau-b
├── 名义 × 名义
│   ├── 期望频数 ≥ 5 → 卡方检验 + Cramer's V
│   ├── 期望频数 < 5 → Fisher 精确检验
│   └── 配对分类变量 → McNemar 检验
├── 连续 × 二分类
│   └── 点二列相关 rpb
└── 连续 × 有序 (3+ 级)
    └── Spearman ρ 或 Eta 系数
```

## 第二步C：预测结果

```
因变量 (DV) 类型？
├── 连续变量
│   ├── 1 个 IV → 简单线性回归
│   ├── 多个 IV → 多元线性回归
│   ├── 需要控制变量 → 层次回归
│   ├── IV 间高相关 → 岭回归 / 删除共线变量
│   └── 非线性关系 → 曲线估计 (二次/三次)
├── 二分类变量
│   └── 逻辑回归
├── 有序多分类变量 (如 满意/一般/不满意)
│   └── 有序逻辑回归 (需检验平行线假设)
└── 名义多分类变量 (3+ 无序类别)
    └── 多项逻辑回归
```

### 回归前提检查

| 检查项 | 方法 | 标准 |
|--------|------|------|
| 线性关系 | 散点图 | 应呈线性 |
| 正态性 | 残差 Q-Q 图 | 点在对角线附近 |
| 等方差性 | 残差 vs 预测值图 | 无漏斗形 |
| 独立性 | Durbin-Watson | 1.5-2.5 |
| 无多重共线性 | VIF | < 10 (理想 < 5) |
| 样本量 | N ≥ 50 + 8×IV 数 | EPV ≥ 10 (逻辑回归) |

## 第二步D：降维 / 结构发现

```
你想做什么？
├── 探索潜在因子结构 → 探索性因子分析 (EFA)
│   前提: KMO > .70, Bartlett 显著
├── 验证已有理论结构 → 验证性因子分析 (需 AMOS/Mplus)
├── 测量量表信度 → Cronbach's α (目标 > .70)
│   其他模型: 折半信度、Guttman、平行模型
├── 将样本分类 → 聚类分析
│   ├── 已知分类数 + 大样本 + 连续变量 → K-Means
│   ├── 不确定分类数 + 小样本 → 层次聚类 (Ward)
│   ├── 混合变量类型 + 自动确定分类数 → TwoStep
│   └── 需要距离/相似度矩阵 → 先做距离分析 (Analyze → Correlate → Distances)
├── 已知分组，找区分变量 → 判别分析
├── 两个分类变量的关系可视化 → 对应分析
├── 分类变量的非线性降维 → 最优尺度分析 (CATPCA)
└── 检验量表结构效度 → EFA + 信度分析组合
```

## 第二步E：因果路径分析

```
研究设计是什么？
├── X 通过 M 影响 Y (中介效应)
│   ├── 逐步法 (Baron-Kenny) → 需要三步回归
│   └── Bootstrap 法 (推荐) → PROCESS 宏，5000 次重抽样
│       判断: 间接效应 95% CI 不含 0 则显著
│
├── X 对 Y 的影响因 Z 而异 (调节效应)
│   ├── 对 X 和 Z 做中心化
│   ├── 计算交互项 XZ
│   └── 层次回归: Step1=X+Z, Step2=XZ
│       判断: ΔR² 显著则调节效应成立
│
└── 同时有中介和调节 (有调节的中介 / 有中介的调节)
    └── 需要 PROCESS 宏 (model 7/8/14/58/59 等)
```

## 第二步F：特殊数据类型

```
你的数据有什么特殊性？
├── 生存/时间数据
│   ├── 比较各组生存率 → Kaplan-Meier + Log-Rank
│   ├── 多因素影响生存 → Cox 比例风险回归
│   └── 时变协变量 → Cox w/ Time-Dependent Covariates
│
├── 计数数据 (次数/频率)
│   ├── 方差 ≈ 均值 → 泊松回归 (GENLIN)
│   ├── 方差 > 均值 (过离散) → 负二项回归
│   └── 零膨胀 → 零膨胀泊松/负二项
│
├── 嵌套/多层数据 (学生在学校/患者在医院)
│   ├── 随机截距模型 → 混合线性模型 (MIXED)
│   ├── 随机截距+斜率 → MIXED + RANDOM INTERCEPT + slope
│   └── 非正态 DV + 嵌套 → 广义线性混合模型 (GENLINMIXED)
│
├── 诊断/筛查评估
│   └── ROC 曲线 → AUC、敏感度、特异度、最佳截断值
│
├── 小样本 / 非正态
│   └── Bootstrap 方法 → SPSS 内建 BOOTSTRAP 命令
│
├── 缺失数据
│   ├── 缺失 < 5% → 列表删除
│   ├── 缺失 5-25% → 多重插补 (MULTIPLE IMPUTATION)
│   └── 缺失机制分析 → MVA + Little's MCAR 检验
│
├── 调查使用了复杂抽样
│   └── 复杂抽样分析 (CSPLAN + CSGLM)
│
└── 分类预测/决策规则
    ├── 需要可视化规则 → 决策树 (CHAID/CRT/QUEST)
    └── 非线性复杂预测 → 神经网络 (MLP/RBF)
```

## 第二步G：质量控制 (工业/制造)

```
你需要什么类型的控制图？
├── 计量数据 (连续测量)
│   ├── 均值+极差图 → X-bar R Chart
│   └── 均值+标准差图 → X-bar S Chart
├── 计数数据 (合格/不合格)
│   ├── 不合格数 → p Chart / np Chart
│   └── 缺陷数 → c Chart / u Chart
├── Pareto 图 → 分析主要缺陷类型
└── 运行图 → 检测趋势
```

## 效应量速查

| 分析类型 | 效应量指标 | 小 | 中 | 大 |
|---------|-----------|---|---|---|
| t 检验 | Cohen's d | 0.2 | 0.5 | 0.8 |
| ANOVA | η² / 偏 η² | .01 | .06 | .14 |
| ANOVA | f | 0.10 | 0.25 | 0.40 |
| 相关 | r | .10 | .30 | .50 |
| 回归 | f² | .02 | .15 | .35 |
| 回归 | R² | .02 | .13 | .26 |
| 卡方 | Cramer's V | .10 | .30 | .50 |
| 逻辑回归 | OR | 1.5 | 3.0 | 9.0 |

## 样本量参考 (α=.05, Power=.80)

| 分析类型 | 最小样本量 | 建议样本量 |
|---------|-----------|-----------|
| t 检验 (中效应, 每组) | 64 | 80+ |
| ANOVA 3 组 (中效应) | 159 | 180+ |
| 相关 (中效应) | 85 | 100+ |
| 多元回归 5 IV (中效应) | 92 | 120+ |
| 因子分析 | 题项数 × 10 | 题项数 × 15-20 |
| 逻辑回归 | EPV ≥ 10 | EPV ≥ 20 |

EPV = Events Per Variable，即每个自变量至少需要 10 个阳性案例
